Добавление пользователей

Чтобы добавить нового пользователя, щелкните Пользователь -> Добавить в панели инструментов.

На закладке Контакт введите имя пользователя, его электронный адрес и другую необходимую персональную информацию: адреса, телефоны и т.д. Перейдите на закладку Пользователь для настройки основных свойств, используемых при подключении пользователя к аккаунту:

  1. Код (логин имя) используется в форме логина вместе с Паролем при подключении пользователя к аккаунту.
  2. Если вы хотите, чтобы пользователь получал email уведомления о различных событиях (обновление файлов в папках, назначения на задания и т.д.) включите флажок "Пользователь может получать почтовые уведомления" и выберите желаемый формат сообщений (HTML или Текстовый).
  3. Язык: выберите язык пользовательского интерфейса.
  4. Часовой пояс и Летнее время: выберите часовой пояс, в котором находится пользователь, чтобы дата/время событий в WebAsyst сервисах отображалось верно. Включите флажок "Летнее время", если в часовом поясе пользователя осуществляется переход на летнее/зимнее время.
  5. Первая страница: выбранная страница будет открываться первой при каждом новом входе пользователя в аккаунт.
  6. Статус: Активный/Неактивный. Пользователям со статусом "Неактивный" доступ в аккаунт заблокирован.

ВНИМАНИЕ: Код пользователя (логин) присваивается единожды при добавлении пользователя, и не может быть в дальнейшем изменен.

Настройка права доступа

Щелкните по закладке Доступ. Включите флажок для тех сервисов и функций, к которым вы хотите разрешить доступ данному пользователю. Для некоторых сервисов возможна настройка доступа на уровне данных. Например, для сервиса WebAsyst Проекты вы можете настроить доступ к каждому проекту на уровне чтения, записи или полный доступ. Аналогично можно настраивать доступ к папкам для таких сервисов как Файлы, Контакты, Блокнот и т.д.

Закладка Доступ позволяет настраивать персональные права доступа. Однако, здесь же, в колонке Групповой отображаются права доступа, унаследованные от тех групп, в которые включен пользователь. Если у пользователя есть групповые права на тот или иной сервис или функцию, то ему не обязательно предоставлять к ним персональный доступ. Управление группами

Часто необходимо предоставить одинаковые права доступа нескольким пользователям. Для этого необходимо использовать инструмент создания групп пользователей и управления групповыми правами. Такие возможности существенно упрощают настройку и использование всех сервисов WebAsyst, особенно при совместной работе с одним аккаунтом большого количества пользователей, например из различных отделов и подразделений вашей организации.

Чтобы не настраивать права доступа каждому индивидуально, создайте “группу пользователей”, настройте для нее права доступа, а потом добавьте необходимых пользователей в эту группу. Все эти пользователи унаследуют групповые права доступа.

Пользователь может входить в несколько групп, а также иметь персональные права доступа. В итоге его права доступа будут объединением прав всех групп, в которые он включен, а также его персональных прав.

Для добавления новой группы, щелкните Группа -> Добавить в панели инструментов.

Для изменения названия группы, состава пользователей и прав доступа для всей группы, щелкните Группа -> Изменить в панели инструментов.

Для удаления группы, щелкните Группа -> Удалить.

ПРИМЕЧАНИЕ: При удалении группы, сами пользователи включенные в эту группу не удаляются.
